# Hotel Management System

## Features

- User Management
- Role Management
- Department Management
- Manager Management
- Employee Management
- Customer Management
- Room Management
- Room Status Management
- Bed Type Management
- Room Type Management
- Booking Management
- Payment Management
- Dashboard

## Technologies

- Laravel
- Inertia.js
- Tailwind CSS

## Installation

Install dependencies

```bash
composer install
npm install
```

## Run the development server

Migrate and seed the database

```bash
php artisan migrate
php artisan db:seed
```

Run the development server

```bash
composer run dev
```
